# TripShare 🧳

**หารค่าใช้จ่ายกับเพื่อน** — แอปหาร Split bill สำหรับกลุ่มทริป รันใน browser ได้เลย ไม่ต้องมี backend

## ✨ Features

- **CRUD ครบ** — สร้าง / ดู / แก้ไข / ลบ ทริป, สมาชิก, และรายการค่าใช้จ่าย
- **แนบรูปภาพ** — แนบสลิปหรือใบเสร็จได้ทุกรายการ (เก็บใน localStorage)
- **คำนวณยอดคงค้าง** — สรุปว่าใครต้องโอนให้ใครเท่าไหร่ แบบเส้นทางสั้นที่สุด
- **สถิติ** — ดูยอดแยกหมวดหมู่และยอดที่แต่ละคนจ่าย
- **หลายทริป** — จัดการหลายทริปพร้อมกันได้
- **ไม่ต้อง login** — ข้อมูลทั้งหมดเก็บใน localStorage ของ browser

## 🚀 Deploy บน GitHub Pages

1. Fork หรือ push ไฟล์ทั้งหมดไปที่ repo GitHub ของคุณ
2. ไปที่ **Settings → Pages**
3. เลือก Branch: `main` และ Folder: `/ (root)`
4. กด **Save** — รอสักครู่แล้วจะได้ URL ของแอป

## 📁 โครงสร้างไฟล์

```
index.html   — หน้าหลัก HTML
style.css    — สไตล์ทั้งหมด
script.js    — logic ทั้งหมด
README.md    — ไฟล์นี้
```

## 📝 วิธีใช้งาน

| action | วิธี |
|--------|------|
| สร้างทริป | กดชื่อทริปด้านบน → สร้างทริปใหม่ |
| แก้ไข / ลบทริป | กดไอคอน ✏️ ข้างทริป |
| เพิ่มสมาชิก | กด `+` ในแถวสมาชิก |
| แก้ไข / ลบสมาชิก | กดชื่อสมาชิก (แก้ไขชื่อ) หรือเปิดแก้ไขทริป (ลบ) |
| เพิ่มรายจ่าย | กดปุ่ม **+** ตรงกลางแถบล่าง |
| ดู / แก้ไข / ลบรายจ่าย | กดที่รายการ |
| แนบรูป | เปิดฟอร์มเพิ่มรายจ่าย → แตะกล่องรูป |
| ดูยอดคงค้าง | แถบ **ยอดคงค้าง** ล่างขวา |
| บันทึกว่าโอนแล้ว | กดปุ่ม "โอนแล้ว ✓" |

## ⚠️ หมายเหตุ

- รูปภาพถูกเก็บเป็น base64 ใน localStorage — หากแนบรูปหลายใบขนาดใหญ่อาจใกล้ limit 5–10 MB ของ browser
- ข้อมูลจะหายเมื่อ clear browser data — แนะนำ export ด้วยตัวเองหากต้องการสำรอง
